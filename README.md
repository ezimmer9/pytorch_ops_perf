# pytorch_ops_perf

Run the app.py:

```shell
python app.py
```

there is 2 flags:
`--no-cmake` - this flag disable the cmake part.
`--no-make` - this flag disable the make part.


if you already have the cmake and the make parts and you wand just run the execute part:

```shell
python app.py --no-cmake --no-make
```
